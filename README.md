# 🔥 HTML File Converter

A simple browser-based file converter using only HTML + JavaScript.

## Features
- 🖼️ Convert images to PNG
- 🎵 Upload and play audio (MP3 download link)
- 🎬 Upload and play video (MP4 download link)
- 🔗 Instant preview + download links

## How it works
- Uses Canvas API for image conversion
- Uses Blob URLs for audio/video playback and downloads
- No backend required

## Run
Just open `index.html` in your browser.

## Limitations
- Audio/video are not truly re-encoded (browser limitation)
- Works fully client-side
